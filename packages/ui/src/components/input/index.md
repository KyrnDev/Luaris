# Input

Notes:
- Support prefix/suffix text and/or icons.
- Support for different types of input (text, password, email, etc).
- Support for validation states (error, warning, success).
- Support for disabled and read-only states.
- Support for different sizes (2xs, xs, sm, md, lg, xl, 2xl, 3xl).
- Support for clearable input.
- Support for auto-focus and focus styles.
- Support for auto-complete and suggestions.
